INVITACIÓN DIGITAL XV AÑOS - JULIA AIDÉ

Archivos incluidos:
- index.html
- styles.css
- script.js

Cómo personalizar:
1. Ubicación en Maps:
   Abre script.js y reemplaza:
   GOOGLE_MAPS_URL
   GOOGLE_MAPS_EMBED

   Puedes usar este formato:
   const GOOGLE_MAPS_URL = "https://maps.google.com/?q=TU%20DIRECCION";
   const GOOGLE_MAPS_EMBED = "https://www.google.com/maps?q=TU%20DIRECCION&output=embed";

2. Confirmación por WhatsApp:
   Abre script.js y cambia el número en WHATSAPP_CONFIRM_URL.
   Ejemplo para México:
   https://wa.me/5213312345678?text=Hola%2C%20confirmo%20mi%20asistencia%20a%20los%20XV%20a%C3%B1os%20de%20Julia%20Aid%C3%A9.%20%F0%9F%A6%8B

3. Música:
   Agrega un archivo MP3 en esta carpeta y nómbralo musica.mp3.
   Nota: algunos celulares bloquean la reproducción automática hasta que la persona toque la pantalla.

4. Código de vestimenta:
   La invitación dice: Semi formal / Favor de no vestir morado 💜.
   Si “No morado” no era para el dress code, cambia esa línea en index.html.

Para publicar:
- Sube la carpeta a Netlify, Vercel, GitHub Pages o tu hosting.
- También puedes comprimir los archivos y enviárselos a quien vaya a subir la web.
