# Invitación web de Stefan Alberto

## Abrir
Haz doble clic en `index.html`.

## Personalizar lugar, mapa y WhatsApp
Abre `script.js` y edita el bloque `invitationConfig` al inicio.

## Agregar PNG de jugadores
Coloca imágenes PNG con fondo transparente en:

- `assets/jugadores/haaland.png`
- `assets/jugadores/cristiano.png`
- `assets/jugadores/mbappe.png`
- `assets/jugadores/lamine.png`
- `assets/jugadores/brian.png`

Se recomienda que tengan proporción vertical, buena resolución y espacio alrededor del cuerpo.

## Fuente Monument Extended
El CSS ya intenta usar `Monument Extended` si está instalada. Para usar un archivo web con licencia:

1. Crea `assets/fonts/`
2. Agrega `MonumentExtended.woff2`
3. Descomenta el bloque `@font-face` al inicio de `styles.css`

No se incluye el archivo de la fuente.

## Música o ambiente
Agrega un MP3 llamado:

`assets/ambiente-estadio.mp3`

El botón de sonido funcionará después de que el visitante lo pulse.

## Publicar
Puedes subir la carpeta completa a Netlify, Vercel, GitHub Pages o al hosting de tu preferencia.
